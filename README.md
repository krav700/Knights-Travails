# Knights-Travails
## Simple program utilizing breadth first search and graphs to calculate the shortest path for a knight on a chess board from position A(x,y) to B(x,y).

## Example
**Input:**
```
knightMoves([0, 0], [7, 7]);
```
**Output:**
```
[ 0, 0 ]
[ 1, 2 ]
[ 2, 4 ]
[ 3, 6 ]
[ 5, 7 ]
[ 6, 5 ]
[ 7, 7 ]
```
